MisterHouse Installation

Here is the main mh documention, a mh FAQ, an announcement list of updates, and the installation instructions.

All of these files are included in the mh/docs directory of the misterhouse_src zip (Updated to version 2.28 on 09/09/00):

    • If using a Windows:
      • misterhouse_src_228.zip: The perl source files and documentation. About 2 Meg. If you have perl installed, this is all you need.
      • misterhouse_win_228.zip: The Windows compiled file. About 2 Meg. This will run stand alone without perl installed on Windows 95,98, NT, or 2K. Note: the above misterhouse_src_228.zip is also required, as it has documentation and support files:
    • If using a Unix, use either of the following, each about 2 Meg:
      • misterhouse-2.28.tar.gz:
      • misterhouse-2.28-1.noarch.rpm:
    • You can get older versions of misterhouse on the Soureforge download page

    You can find this page at either of the following:

    • http://misterhouse.net/download
    • http://misterhouse.sourceforge.net
