Where to Download the latest version of MisterHouse

Version 2.100 released on 10/02/2005:

  • If using a Windows:
    • misterhouse_src_2100.zip (previous stable release: 2.99): The Perl source files and documentation. About 7 Meg. If you have Perl installed, this is all you need.
    • misterhouse_win_2100.zip (previous stable release: 2.99) : The Windows compiled file. About 6 Meg. This will run stand alone without Perl installed on Windows 95,99, NT, or 2K. Note: the above misterhouse_src_2100.zip is also required, as it has documentation and support files.
  • If using a Unix, use either of the tar.gz or .rpm files here, each about 6 Meg:
    • misterhouse-2.100.tar.gz (previous stable release: 2.99)
    • misterhouse-2.100-1.noarch.rpm (previous stable release: 2.99)
  • If using a recent Linux (ls -l /lib/libc.so* and look ver a version > 2.2.95, like on RH 8 or higher) you can also try this compiled version:
    • misterhouse-2.100.linux.zip (previous stable release: 2.99) : Note: one of the above src, tar, or rpm files are also required, as they have the documentation and support files.

Between formal releases (every 2->6 weeks), we have informal test releases, available here. Changes in the test release are listed here.

You can get older versions of MisterHouse on the Soureforge download page

A large (30 meg) file of optional files (mainly sound files) is here: misterhouse_misc.zip

If you are having problems downloading from the above servers, you can also get the above files from these locations:

  • The sourceforge upload http server
  • The sourceforge ftp server
  • The misterhouse.net/public directory (a slower link)

All the code is also available at misterhouse.net/mh and there is a MisterHouse CVS repository (used by co-developers) complements of our friends at SourceForge.


Subscribe to the misterhouse-anounce mailing list to get an email summarizing changes of at release.
