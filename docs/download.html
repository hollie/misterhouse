Where to Download the latest version of MisterHouse

Version 2.104 released on 11/04/2007:

  • If using a Windows:
    • misterhouse_src_2104.zip (previous stable release: 2.103): The Perl source files and documentation. About 9 Meg. If you have Perl installed, this is all you need.
    • misterhouse_win_2104.zip (previous stable release: 2.103) : The Windows compiled file. About 6 Meg. This will run stand alone without Perl installed on Windows 95,100, NT, or 2K. Note: the above misterhouse_src_2104.zip is also required, as it has documentation and support files.
  • If using a Unix, the tar file is about 6 Meg:
    • misterhouse-2.104.tar.gz (previous stable release: 2.103)

Between formal releases (every 2-4 months), you can pick up the latest updates from the SVN repository (more info on our wiki ) or in this zip file, compiled nightly. Changes in the test release are listed in the SVN log.

You can get older versions of MisterHouse on the Soureforge download page

A large (30 meg) file of optional files (mainly sound files) is here: misterhouse_misc.zip

All the code is also available at local.misterhouse.net/mh


Subscribe to the misterhouse-anounce mailing list to get an email summarizing changes when a new release is available.
